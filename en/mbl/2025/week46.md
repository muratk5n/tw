LLMs are just another type of neural network. If the "weights (a list
of numbers)" are available, anyone can run an LLM on their own
machine, no outside connection needed. Training a large NN takes a lot
of resources, but using a trained NN requires less.

Llama is an open source tool / site, it hosts open-data LLMs. Install
the tool and it lets you choose from a list, pick one, it downloads,
installs the weights. Then via the prompt one can talk to a local LLM
neural network. With a few hundred GB of RAM, a good graphics card,
even a large model should run fine.
